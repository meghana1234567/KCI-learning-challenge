OWASP is the emerging standards body for web application security. In particular they have published the OWASP Top 10,which describes in detail the major threats against web applications. 
The Web Application Security Consortium (WASC) has created the Web Hacking Incident Database (WHID) and also produced open source best practice documents on web application security. 

Open Web Application Security Project (OWASP):

OWASP is an online community that produces freely-available articles, methodologies, documentation, tools, and technologies in the field of "web application security".
